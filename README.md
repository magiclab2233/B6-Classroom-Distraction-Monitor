# B6-基于人脸关键点的上课走神监测系统

## 项目简介

基于人脸关键点的上课走神监测系统是一款面向智慧课堂场景的 AI 教育辅助工具。项目旨在通过计算机视觉技术实时监测学生课堂上的面部表情与头部姿态，判断学生是否处于专注或走神状态，帮助教师及时了解课堂注意力分布，提升教学效果。

## 功能特点

- **实时人脸检测**：通过摄像头捕捉学生面部关键点，无需额外硬件设备。
- **专注度分析**：基于人脸关键点变化（如视线方向、头部转动、眨眼频率等）评估学生专注状态。
- **课堂走神预警**：当检测到学生长时间处于走神状态时，系统自动记录并提示。
- **多平台部署**：支持在教室一体机、教师端电脑等多种设备上运行。

## 技术架构与实现原理

### 整体架构

```
教室摄像头 → 人脸检测模型 → 关键点提取 → 姿态/表情分析 → 专注度评分 → 可视化报表
```

### 核心技术

- **MediaPipe Face Mesh**：实时提取 468 个面部关键点，为后续分析提供精确数据。
- **OpenCV**：视频流捕获与图像预处理。
- **Python**：算法实现与数据分析核心语言。
- **注意力评估算法**：结合头部姿态（欧拉角）、眼部纵横比（EAR）、视线方向等多维度特征，综合计算专注度得分。

## 安装与运行说明

### 环境要求

- Windows / macOS / Linux 系统
- Python 3.8+
- 摄像头设备

### 安装步骤

1. 安装 Python 环境（参考 `1-python安装` 目录中的安装包）。
2. 安装 VSCode 编辑器（参考 `2-vscode安装` 目录中的安装包）。
3. 安装课程所需 AI 工具（参考 `3-couseAI安装` 目录中的安装包）。
4. 克隆项目代码：
   ```bash
   git clone https://github.com/magiclab2233/B6-Classroom-Distraction-Monitor.git
   cd B6-Classroom-Distraction-Monitor
   ```
5. 根据项目内部说明配置运行环境并启动监测程序。

## 使用场景

- **中小学智慧课堂**：实时监测全班学生专注度，帮助教师调整授课节奏。
- **在线教育直播**：分析学员听课状态，生成课后学习报告。
- **AI 教学演示**：作为计算机视觉在教育领域应用的教学案例。
